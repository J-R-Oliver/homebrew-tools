## How do I install these formulae?

`brew install j-r-oliver/tools/<formula>`

Or `brew tap j-r-oliver/tools` and then `brew install <formula>`.

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
